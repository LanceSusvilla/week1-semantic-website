# 🍽️ GILLE Restaurant

> **Week 1 Semantic HTML5 Website Project**

A simple multi-page restaurant website created using **Semantic HTML5** as part of our IPT-101 assignment.

---

# 📖 Project Overview

GILLE Restaurant is a static website that introduces a fictional restaurant and its signature dishes.

The website allows visitors to:

- 🏠 Browse the Home page
- 🍽 View the Menu
- 📖 Learn About the restaurant
- 📞 Contact the restaurant through a contact form

---

# 📂 Website Pages

| Page | Description |
|------|-------------|
| 🏠 Home | Welcome page featuring today's top picks |
| 🍽 Menu | Displays signature dishes and restaurant price list |
| 📖 About | Story, mission, vision, and restaurant information |
| 📞 Contact | Contact form for customer inquiries |
